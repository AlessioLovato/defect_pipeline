Put your Detectron2 model files here.

Default names used by launch:
- `maskrcnn-height-curv-augmented-16/config.yaml` for the Detectron2 config.
- `maskrcnn-height-curv-augmented-16/model_final.pth` for weights.

You can override paths at launch:
- `model_config_path:=/absolute/path/to/config.yaml`
- `model_weights_path:=/absolute/path/to/weights.pth`
